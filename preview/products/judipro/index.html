JUDIPRO · Julia inversion framework

Wave-physics inversion in Julia, production-grade.

JUDIPRO pairs JUDI — the open-source Julia Devito Inversion framework from Georgia Tech’s SLIM group — with DevitoPRO as a bring-your-own-license backend, bringing performance portability and scalability to JUDI’s operator-algebra modeling, inversion and machine-learning workflows.

What the DevitoPRO backend adds

From research prototype to industry-scale 3D

JUDI translates wave-based algorithms into fast, scalable Julia code; DevitoPRO supplies the production-grade propagators underneath.

GPU code generation

CUDA, HIP and SYCL backends for JUDI’s solvers — the same Julia code runs across AMD, Intel and NVIDIA GPUs.

Large-model memory

Automatic source/receiver expanding box for forward and adjoint solves, plus asynchronous wavefield serialization with lossy/lossless compression.

Multi-device solves

The DevitoPRO Decoupler does single-shot domain decomposition over multiple devices or NUMA domains while keeping a serial Julia process.

Performance portability

State-of-the-art performance and scalability for long-offset FWI, high-frequency RTM and full-wavefield imaging — an easy path from prototype to production.

Built for research-to-production

Where JUDIPRO shines

Probabilistic FWI

Variational inference with conditional normalizing flows (WISE) for uncertainty quantification in velocity models.

Carbon storage monitoring

Compressive time-lapse seismic monitoring of CO₂ plumes for carbon capture and storage (CCS).

Learned inversion

Fourier Neural Operators and coupled inversion for permeability estimation and CO₂ plume forecasting.

Serverless cloud imaging

Event-driven seismic imaging on cloud infrastructure — scalable, cost-effective, no server management.

JUDI’s modeling operators integrate directly into neural networks for physics-augmented deep learning and differentiable programming — combining Devito’s computational power with Julia’s flexibility for PDE-constrained optimization like FWI and LS-RTM.
